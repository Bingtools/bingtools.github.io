Bingo ToolBox 使用教程

从安装到日常使用的完整教程，新手建议按顺序阅读

Bingo ToolBox — 个人 Maya 工具箱插件

EXE 安装包支持自定义安装路径（默认 D:/Maya_Tool/Bingo_Tools），用分类、收藏、快捷键和工具注册信息管理脚本，适合把零散 Python/MEL 工具集中起来。

  • 内置动画、绑定、模型、特效、其他等常用工具分类
  • 支持工具图标、收藏星标、快捷入口和自定义菜单
  • 可把 Python / MEL / EXE / 文件夹统一登记管理
  • 已按 Maya 2020–2027 全版本兼容方向整理

1. 安装方式

双击运行 EXE 安装包，安装程序会弹出路径选择窗口，默认路径为 D:/Maya_Tool/Bingo_Tools，可点击「浏览」自定义安装路径。工具文件直接放入所选目录，不会多套一层版本文件夹。Maya 启动时会自动通过 Documents/maya/modules/Bingo_ToolBox.mod 指向实际安装路径。

默认安装路径: D:\Maya_Tool\Bingo_Tools
Module 入口: Documents\maya\modules\Bingo_ToolBox.mod
备份: 安装目录_backup_时间戳
主入口: scripts\maya_plugin_manager_ui.py

2. 打开工具箱

安装完成后重新启动 Maya，在 Maya 顶部菜单栏找到 Bingo 菜单，点击 ToolBox 即可打开工具箱主界面。

Maya 菜单栏 → Bingo → ToolBox

3. 工具简介

Bingo ToolBox 用来把制作中常用的动画、绑定、模型、特效和其他脚本集中到一个面板里。每个工具都可以配置名称、说明、图标、分类和执行命令。

工具类型: Animation / Rigging / Model / FX / Other

4. 使用方法

打开 ToolBox 后先选择左侧分类，再点击工具卡片即可执行。右键分类区域可新建工具，填入名称、简介、图标和命令。

1. 选择分类或新建分类
2. 右键新建工具并填写说明
3. 粘贴 Python/MEL/EXE 启动命令
4. 选择图标并保存
5. 点击工具卡片运行

5. 注册脚本工具

右键分类区域 →「新建工具」，填写名称、描述，粘贴 Python 或 MEL 代码，指定图标和快捷键后保存。

import maya.cmds as cmds
ctrls = cmds.ls("*_ctrl", type="transform")
cmds.select(ctrls)

6. 自定义菜单

进入工具箱设置 → 菜单编辑器，拖拽工具到菜单树中即可在 Maya 主菜单栏创建层级菜单。

Bingo/
  ├── Animation/
  ├── Rigging/
  └── Utility/

7. 卸载方式

运行安装目录下的 uninstall_Bingo_ToolBox.cmd 移除 Maya module 入口，再手动删除工具目录。

uninstall_Bingo_ToolBox.cmd
